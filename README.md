This project focuses on analyzing Indian airline data to uncover insights related to flight prices, airline performance, travel duration, stops, routes, and customer travel patterns.
The analysis helps identify trends in airfare pricing, busiest routes, preferred airlines, and factors affecting ticket costs.

Objective

The main objectives of this project are:

Analyze airline ticket pricing trends
Compare different airlines based on fare and duration
Identify factors affecting flight prices
Understand route popularity and stop patterns
Create meaningful visualizations for better decision-making
